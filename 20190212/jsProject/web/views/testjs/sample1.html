자바스크립트 작성 위치와 작성 방법

inline방식 시작 태그에 자바스크립트가 제공하는 이벤트 핸들러(on이벤트명)를 사용하여
자바 스크립트 소스를 연결할 수 있다. on이벤트 속성은 자바스크립트가 제공하며, 모든 태그에 사용할 수 있다.

internal방식 : 자바스크립트 소스를 script 태그안에 function(함수) 단위로 작성하는 방법

script태그의 작성 위치는 head body html 밖 어디든 작성가능하다.
한 문서 안에 여러 번 사용해도 된다.html태그에서 자바스크립트 함수를 실행하는 방법은 시작태그에 on이벤트 속성을 추가하고
속성값에 실행할 함수명()을 사용한다.
